# Extranet Documental Personal
## Configuración
1. Crea cuenta gratuita en https://supabase.com
2. Crea proyecto y bucket 'documentos'
3. Crea tabla 'roles' con columnas: id_usuario(text), nivel(integer), baneado(boolean)
4. Copia SUPABASE_URL y SUPABASE_ANON_KEY
5. Pega las credenciales en script.js
6. Crea usuario admin inicial
7. Sube los archivos a GitHub y activa Pages
